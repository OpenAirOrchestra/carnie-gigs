Gig Calendar for The Carnival Band

== Description ==

A gig calendar specifically tailored for The Canival Band.

== Installation ==

1. Upload `carnie-gigs.zip` to the `/wp-content/plugins/` directory and unzip.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the shortcode [carniegigs] to a page to list all gigs.
4. Add the shortcode [carniegigs time="past"] to a page to list past gigs.
5. Add the shortcode [carniegigs time="future"] to a page to list future gigs.

6. Add the shortcode [gigattendance gigid="12"] to a page to display gig 
   attendance widget for the gig with the id "12".
7. Add the shortcode 
	[gigattendance gigid="12"] Alice, Bruce, Chen [/gigattendance]
   to display a minimal gig attendance widget for the gig with the id "12" 
   and display the attendees as Alice, Bruce, and Chen. This form of the
   gig attendance widget doesn't need to hit the gig database to display.
